# Task Planner

A simple command-line task manager written in python.


## Features

- Add tasks
- Remove tasks
- Search tasks
- Edit tasks 
- Mark tasks as Done / Pending
- Sort tasks
- Save tasks to CSV
- Load tasks automatically


## Technologies

- PYthon
- CSV
- OOP
- Git


## Project Structure
 
```
task.py
manager.py
storage.py
main.py
```

## How to Run

```bash
python main.py
```
