# Programming Planner

A simple command-line task manager written in Python.

## Features

- Add tasks
- Remove tasks
- Search tasks
- Edit tasks
- Mark tasks as Done / Pending
- Sort tasks
- Save tasks to CSV
- Automatically load tasks

## Technologies

- Python
- CSV
- OOP
- Git

## Project Structure

- task.py
- manager.py
- storage.py
- main.py

## How to Run

```bash
python main.py
```
